¿Qué es entonces una base de datos - DB?

Base de Datos: Conjunto de información perteneciente a un mismo contexto, ordenada de modo sistemático para su posterior recuperación, análisis y/o transmisión.

En otras palabras, una base de datos es una colección de archivos interrelacionados. Algo importante es que hay que tener cuidado en el diseño de la base de datos, para que los archivos interrelacionados que la integran se repitán lo menor posible y cumpla con las necesidades de la empresa o entidad.

¿En qué casos es conveniente usar bases de datos?

Es conveniente usar bases de datos cuando se requiere almacenar gran número de información de una forma organizada para su futura consulta, realizar busquedas, ingresar nuevos datos, etc; además todo de forma rápida y simple desde una computadora. Tambien se usan las bases de datos cuando la información manejada tiene archivos interrelacionados y es requerida por varios usuarios o aplicaciones.

¿Qué es un sistema de gestión de base de datos - DBMS?

DBMS (Data Base Management System) : Un sistema gestor de base de datos es un conjunto de programas que permiten el almacenamiento, acceso, modificación y extracción de información de una colección de datos interrelacionados, (Base de Datos).

Enlista y explica las funciones y responsabilidades que tienen los DBMS

Funciones y Responsabilidades DBMS:

1. Crear y organizar la base de datos: Define estructuras para el almacenamiento de información.

2. Redundancia e inconsistencia de los datos: Ajusta los diferentes formatos y los programas duplicados en varios sitios (archivos), ya que la redundancia aumenta los costos de almacenamiento y acceso.

3. Acceso a los datos: Establece y mantiene las trayectorias de acceso a la base de datos de tal forma que los datos puedan ser accesados rápidamente.

4. Manejar los datos de acuerdo a las peticiones de los usuarios: Gestiona grandes bloques de información y provisiona mecanismos para la gestión de la información

5. Registra el uso de la base de datos.

6. Aislamiento de los datos.

7. Control de concurrencia: Los sistemas permiten que múltiples usuarios actualicen los datos simultáneamente. En un entorno así, la interacción de actualizaciones concurrentes puede dar por resultado datos inconsistentes y anomalías, por lo que el control de concurrencia permite controlar la interacción entre los usuarios concurrentes para no afectar la inconsistencia de los datos.

8. Seguridad: Da seguridad a la información almacenada, pese a caídas del sistema o intentos de accesos no autorizados, además no todos los usuarios deben poder acceder a todos los archivos. Tambien consiste en contar con mecanismos que permitan el control de la consistencia de los datos evitando que estos se vean perjudicados por cambios no autorizados o previstos.

9. Integridad: Se deben satisfacer ciertos tipos de restricciones de consistencia.

10. Respaldo y recuperación: Consiste en contar con mecanismos implantados que permitan la recuperación fácilmente de los datos en caso de ocurrir fallas en el sistema de base de datos.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En la materia de Programación Orientada a Objetos hice un proyecto para una farmacéutica, en la que tenía que llevar un inventario de las medicinas, los elementos que contenía la medicina y la cantidad, la cantidad de capsulas, las restricciones de ingestión y la dosis según la edad.

En esté caso hubiera sido conveniente utilizar una base de datos, porque se requiería almacenar gran número de información sobre los médicamentos de una forma organizada para su futura consulta, además se hacían busquedas sobre los medicamentos, se requería ingresar nuevos datos de medicianas, etc. Además todo esto tenía que ser de forma rápida y simple desde una computadora.

Otro punto importante es que los archivos estaban interrelacionados y eran requeridos por varios usuarios o aplicaciones, por ejemplo: el departamento químico de la farmaceutica, las farmacias afiliadas a la farmacéutica, los doctores, los programas que calculaban las dosis, etc.

Es por eso que en ese proyecto hubiera sido conveniente utilizar una base de datos.
