(A - Snacktower)

According to an old legeng, a long time ago Ankh-Morpork residents did something wrong to miss Fortune, and she cursed them. She said that at some time n snacks of distinct sizes will fall on the city, and the residents should build a Snacktower of them by placing snacks one on another. Of course, big snacks should be at the bottom of the tower, while small snacks should be at the top.

Years passed, and once different snacks started to fall onto the city, and the residents began to build the Snacktower.

However, they faced some troubles. Each day exactly one snack fell onto the city, but their order was strange. So, at some days the residents weren't able to put the new stack on the top of the Snacktower: they had to wait until all the bigger snacks fell. Of course, in order to not to anger miss Fortune again, the residents placed each snack on the top of the tower immediately as they could do it.

Write a program that models the behavior of Ankh-Morpork residents.

Input:
The first line contains single integer n (1 ≤ n ≤ 100 000) — the total number of snacks.

The second line contains n integers, the i-th of them equals the size of the snack which fell on the i-th day. Sizes are distinct integers from 1 to n.

Output:
Print n lines. On the i-th of them print the sizes of the snacks which the residents placed on the top of the Snacktower on the i-th day in the order they will do that. If no snack is placed on some day, leave the corresponding line empty.

Example 1:

Input:
3
3 1 2

Output:
3

2 1

Example 2:

Input:
5
4 5 1 2 3

Output:

5 4

3 2 1

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: JavaScript es un lenguaje interpretado, que se lee y se traduce línea a línea al mismo tiempo que se ejecuta el programa, además JavaScript se ejecuta esencialmente a través del navegador web. Por otro lado, Java es un lenguaje compilado que pasa por una JVM (Java Virtual Machine) para ser interpretado y traducido al lenguaje de la máquina.

Resumiendo las diferencias, JavaScript se ejecuta en un navegador web y Java en un maquina virtual.

Similitudes: Tanto JavaScript como Java son lenguajes de programación que pueden implementar el paradigma orientado a objetos y además tienen una sintaxis muy similar.

¿Qué métodos tiene el objeto Date?
(Menciona al menos 5*)

Date.now( ): Devuelve el valor numérico correspondiente a la hora actual.

Date.getDate( ): Devuelve el día del mes. Número entre 1 y 31.

Date.getMonth( ): Devuelve el número de mes del 0 (enero) al 11 (diciembre).

Date.getDay( ): Devuelve el día de la semana. Entre 0 (domingo) y 6 (sábado).

Date.getHours( ): Devuelve la hora actual, desde 0 a 23 (formato 24 horas).

Date.getMinutes( ): Devuelve los minutos actuales, desde 0 a 59.

Date.getSeconds( ): Devuelve los segundos actuales, desde 0 a 59.

Date.getFullYear( ): Devuelve el año con 4 dígitos.

¿Qué métodos tienen los arreglos?
(Menciona al menos 5*)

concat( ): Une dos o más arrays.

indexOf( ): Busca un elemento en el array y devuelve su posición. Busca desde el principio y a partir de la primera posición si no se especifica nada.

join( ): Junta los elementos de un array en una cadena con un separador – opcional.

lastIndexOf( ): Busca un elemento en un array y devuelve su posición. Comienza buscando por el final del array.

pop( ): Borra el último elemento del array y devuelve su contenido.

push( ): Añade nuevos elementos al array y devuelve su nueva longitud.

reverse( ): Invierte el orden de los elementos del array.

shift( ): Elimina el primer elemento del array y devuelve el elemento.

slice( ): Selecciona parte de un array y devuelve su contenido.

sort( ): Ordena los elementos del array. Los ordena como strings.

splice( ): Añade o elimina elementos de un array.

toString( ): Convierte un array en string y devuelve el resultado(entre comas).

unshift( ): Añade elementos al inicio del array y devuelve la nueva longitud.

split( ): Convierte un string en un array.

¿Cómo se declara una variable con alcance local dentro de una función?

Para declara una variable con alcance local dentro de una función se usa var, otra opción es usar let que hace que una variable exista en el bloque en el que existe.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables globales son accesibles desde cualquier lugar de la página, es decir, desde el script de JavaScript donde se han declarado y todos los demás scripts de la página, incluidos los manejadores de eventos. En el caso de las funciones, las variables pueden ser fácilmente modificadas en cualquier otra función y no se puede usar el mismo nombre de la variable para nombrar otra variable dentro de cualquier otra función o bloque.
