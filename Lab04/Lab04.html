¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: JavaScript es un lenguaje interpretado, que se lee y se traduce línea a línea al mismo tiempo que se ejecuta el programa, además JavaScript se ejecuta esencialmente a través del navegador web. Por otro lado, Java es un lenguaje compilado que pasa por una JVM (Java Virtual Machine) para ser interpretado y traducido al lenguaje de la máquina.

Resumiendo las diferencias JavaScript se ejecuta en un navegador web y Java en un maquina virtual.

Similitudes: Tanto JavaScript como Java son lenguajes de programación que pueden implementar el paradigma orientado a objetos, además de tener una sintaxis muy similar.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Date.now( ): Devuelve el valor numérico correspondiente a la hora actual.

Date.getDate( ): Devuelve el día del mes. Número entre 1 y 31.

Date.getMonth( ): Devuelve el número de mes del 0 (enero) al 11 (diciembre).

Date.getDay( ): Devuelve el día de la semana. Entre 0 (domingo) y 6 (sábado).

Date.getHours( ): Devuelve la hora actual, desde 0 a 23 (formato 24 horas).

Date.getMinutes( ): Devuelve los minutos actuales, desde 0 a 59.

Date.getSeconds( ): Devuelve los segundos actuales, desde 0 a 59.

Date.getFullYear( ): Devuelve el año con 4 dígitos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

concat( ): Une dos o más arrays.

indexOf( ): Busca un elemento en el array y devuelve su posición. Busca desde el principio y a partir de la primera posición si no se especifica nada.

join( ): Junta los elementos de un array en una cadena con un separador – opcional.

lastIndexOf( ): Busca un elemento en un array y devuelve su posición. Comienza buscando por el final del array.

pop( ): Borra el último elemento del array y devuelve su contenido.

push( ): Añade nuevos elementos al array y devuelve su nueva longitud.

reverse( ): Invierte el orden de los elementos del array.

shift( ): Elimina el primer elemento del array y devuelve el elemento.

slice( ): Selecciona parte de un array y devuelve su contenido.

sort( ): Ordena los elementos del array. Los ordena como strings.

splice( ): Añade o elimina elementos de un array.

toString( ): Convierte un array en string y devuelve el resultado(entre comas).

unshift( ): Añade elementos al inicio del array y devuelve la nueva longitud.

split( ): Convierte un string en un array.
